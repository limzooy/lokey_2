# LoKey - 요즘 20대의 조용한 취향 코드

20-30대가 조용히 소비하는 패션·빈티지·저도수 라이프스타일 트렌드를 큐레이션하는 미디어 플랫폼

## 프로젝트 개요

- **컨셉**: Quiet Luxury, Low-key Taste
- **목표**: 트렌드 큐레이션 및 분석 콘텐츠 제공
- **수익 모델**: Google AdSense, 제휴 링크, 브랜드 협업

## 기술 스택

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Vercel (권장)

## 시작하기

### 1. 의존성 설치

```bash
npm install
```

### 2. 개발 서버 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

### 3. 빌드

```bash
npm run build
npm start
```

## 페이지 구조

- `/` - 홈페이지
- `/category` - 카테고리 목록
- `/category/fashion` - 패션 카테고리
- `/category/vintage` - 빈티지 카테고리
- `/category/low-alcohol` - 저도수 술 카테고리
- `/post/[slug]` - 개별 포스트
- `/about` - About 페이지
- `/contact` - Contact 페이지
- `/privacy` - 개인정보 처리방침
- `/terms` - 이용약관

## SEO 최적화
